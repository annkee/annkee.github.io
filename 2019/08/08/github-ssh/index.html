centos7 gitlab安装

安装SSH服务，在系统防火墙中打开HTTP和SSH访问

安装Postfix，发送通知电子邮件（邀请团队成员的时候用得上）

下载安装包

下载地址： https://packages.gitlab.com/gitlab/gitlab-ce

根据自己需要进行版本选择。

安装

sudo EXTERNAL_URL="http://ip:端口"=rpm -i rpm包

注意：端口不要和现有的冲突

查看配置文件

vi /etc/gitlab/gitlab.rb

主要看unicorn[‘port’]=8079(端口不要冲突)

重启配置

gitlab-ctl reconfigure

git命令窗口输入

测试：
$ ssh -T git@github.com

出现:Hi annkee! You've successfully authenticated, but GitHub does not provide shell access.sh 说明正常
